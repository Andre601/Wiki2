# Fish Slapper

FishSlapper allows any player with the correct permission to use raw fish as an tool to slap people around the server in a chacotic fashion.
It's quite a simple concept, those with the permission allowing them have the permission to slap can slap anyone with the permission that allows them to be slapped.
Once contact has been made between the fish and the player they are then sent flying away!
(Pufferfish will allow for maximum slapping)
