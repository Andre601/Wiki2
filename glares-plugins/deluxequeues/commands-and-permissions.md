# Commands & Permissions

## Player Commands

| Command | Permission | Description |
| :--- | :--- | :--- |
| /dq | deluxequeues.help | Shows the help menu |
| /dq leave | deluxequeues.leave | Leave your current queue |

## Admin Commands

| Command | Permission | Description |
| :--- | :--- | :--- |
| /dq reload | deluxequeues.admin | Reloads the config |
